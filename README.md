# Repository Advisor

An interactive web application that helps researchers identify suitable research data repositories based on their research needs.

## About

The Repository Advisor is an initiative developed as part of the **CUHK Data Champions Programme**. It helps researchers identify appropriate data repositories through an interactive recommendation system based on repository characteristics and research requirements.

## Project Background

The Repository Advisor was initiated by **Data Matters @ CUHK** as part of the CUHK Data Champions Programme. The project builds upon the repository evaluation matrix originally developed by **Qinqin Zhang** and the **CUHK Data Team**, which served as the foundation for the repository database. Through ongoing collaboration, discussion, and iterative refinement, the repository database, recommendation framework, and interactive prototype have continued to evolve.
